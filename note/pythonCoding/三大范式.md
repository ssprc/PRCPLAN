  一、从 Java 视角看 Python 的关键区别                                                                                                                          
                                                                                                                                                                
  1. 没有类型声明（但有类型提示）                                                                                                                               
                                                            
  # Java                                                                                                                                                        
  private String model;                                                                                                                                         
  private int timeout;                                                                                                                                          
                                                                                                                                                                
  # Python - 直接赋值，不需要声明类型                                                                                                                           
  self.model = model or os.getenv("LLM_MODEL_ID")                                                                                                               
  self.timeout = timeout or int(os.getenv("LLM_TIMEOUT", 60))                                                                                                   
                                                                                                                                                                
  你在代码里看到的 str、List[Dict[str, str]] 这些只是类型提示，不强制，运行时完全不检查。比如 llm_client.py 第28行：                                            
                                                                                                                                                                
  def think(self, messages: List[Dict[str, str]], temperature: float = 0) -> str:                                                                               
                                                                                                                                                                
  这等价于 Java 的方法签名，但 Python 运行时不管你传什么类型。                                                                                                  
                                                                                                                                                                
  2. 没有 new，没有 public/private                                                                                                                              
                                                            
  // Java                                                                                                                                                       
  HelloAgentsLLM llm = new HelloAgentsLLM();                                                                                                                    
                                                                                                                                                                
  # Python - 直接调用类名，就是构造                                                                                                                             
  llm = HelloAgentsLLM()                                                                                                                                        
                                                                                                                                                                
  Python 的 __init__ 就是 Java 的构造方法。self 就是 Java 的 this。                                                                                             
                                                                                                                                                                
  没有真正的 private，约定俗成用 _ 开头表示私有，比如 _parse_output，但其实外部照样能调。                                                                       
                                                            
  3. 没有花括号，靠缩进                                                                                                                                         
                                                            
  // Java                                                                                                                                                       
  if (action.startsWith("Finish")) {                                                                                                                            
      String answer = parse(action);                                                                                                                            
      return answer;                                                                                                                                            
  }                                                                                                                                                             
                                                                                                                                                                
  # Python - 缩进就是代码块                                                                                                                                     
  if action.startswith("Finish"):                                                                                                                               
      answer = parse(action)                                                                                                                                    
      return answer                                                                                                                                             
                                                                                                                                                                
  这是 Python 初学者最容易出错的地方，缩进错了直接报语法错误。

  三、Python 的项目构建 vs Java                                                                                                                                 
                                                                                                                                                                
  这是 Java 开发者最容易懵的部分：
                                                                                                                                                                
  ┌──────────────────────────┬─────────────────────────────────┐                                                                                                
  │           Java           │             Python              │                                                                                                
  ├──────────────────────────┼─────────────────────────────────┤                                                                                                
  │ Maven/Gradle 管依赖      │ pip install -r requirements.txt │
  ├──────────────────────────┼─────────────────────────────────┤                                                                                                
  │ pom.xml / build.gradle   │ requirements.txt                │                                                                                                
  ├──────────────────────────┼─────────────────────────────────┤                                                                                                
  │ src/main/java/... 包结构 │ 没有强制目录规范                │                                                                                                
  ├──────────────────────────┼─────────────────────────────────┤                                                                                                
  │ mvn test                 │ python test_xxx.py 直接跑       │
  ├──────────────────────────┼─────────────────────────────────┤                                                                                                
  │ JVM 运行                 │ 直接 python xxx.py              │
  ├──────────────────────────┼─────────────────────────────────┤                                                                                                
  │ 编译成 .class            │ 解释执行，不需要编译            │
  └──────────────────────────┴─────────────────────────────────┘                                                                                                
                                                            
  这个项目没有 requirements.txt 在根目录，因为每个章节是独立的。要运行第4章的代码：                                                                             
                                                            
  pip install openai python-dotenv serpapi                                                                                                                      
  python ReAct.py

  snippets = [f"[{i+1}] {res.get('title', '')}" for i, res in enumerate(results[:3])]

  List<String> snippets = IntStream.range(0, 3)

      .collect(Collectors.toList());
